# MÄKA Baby CRM

Sistema de Gestión de Clientes y Ventas para MÄKA Baby.

## Características
- Gestión de Clientes (CRUD).
- Registro de Ventas con detalle de productos.
- Generación de Facturas y Etiquetas de Envío (PDF/Impresión).
- Panel de estadísticas mensuales y anuales.
- Diseño responsivo para dispositivos móviles.

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- LocalStorage para persistencia de datos local.

## Despliegue en GitHub Pages
Este proyecto está preparado para ser alojado en GitHub Pages. Simplemente sube los archivos a un repositorio y activa la opción en Settings > Pages.
